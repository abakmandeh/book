# AGIT BOOK API

## Require
- java 8
- maven 2.5.7

## Steps to run
1. Run script from table.sql in recources folder
2. Build the project using
   `mvn clean install`
3. Run using `mvn spring-boot:run`
4. The API is accessible via localhost:8080
